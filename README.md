# bgmprovider

#### 介绍
BGMProvider目标是提供一个完整的GMTLS JAVA实现。

#### 软件架构
BGMProvider 基于 Java Cryptography Architecture(JCA) 框架，
提供一个JCE provider 和 JSSE provider。
